Getting Your JSearch Key (5 min)
1
Go to rapidapi.com → click Sign Up → use Google login (easiest)
2
In the search bar at the top, type JSearch and press Enter
3
Click the result called "JSearch" by letscrape
4
Click "Subscribe to Test" → select the Basic (Free) plan → click Subscribe
5
You'll see your key under "X-RapidAPI-Key" in the right panel — copy it
6
Paste it in the API Keys tab above
Getting Your Groq Key (2 min)
1
Go to aistudio.google.com/app/apikey
2
Sign in with your Google / Gmail account
3
Click "Create API Key" → then "Create API key in new project"
4
Copy the key (starts with gsk_...) → paste in API Keys tab above
Both are completely free. No credit card required for either.
